How It Works

Gaussian Points uses a hybrid Ruby/C++ implementation with low-level OpenGL hooks to render millions of points at interactive framerates.

  • Multi-format Support

    Import point clouds from e57, LAS, PLY, and other industry-standard formats

  • Low-level Optimization

    Direct OpenGL integration using VBO rendering for maximum performance

  • Gaussian Splatting

    Revolutionary rendering technique that creates smooth, photorealistic surfaces from points

  • Powerful Toolset

    Clipping planes, scaling controls, and interactive manipulation tools
